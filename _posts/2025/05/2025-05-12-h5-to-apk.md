# h5打包成apk

1. 替换图标
```
res/mipmap-hdpi/ic_launcher.png
res/mipmap-xhdpi/ic_launcher.png
```

2. 替换网址
assets/c.txt 中的


3. 替换名称
AndroidManifest.xml 中找

4. 打包
- 退出目录
- 点击app
- apk签名->确定

就行了
