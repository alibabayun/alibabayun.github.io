### 前言
一个有、意思的工具, 可以把web页面打包成运行在桌面的应用, 并支持win / mac / linux等平台, 记一下使用过程, 有(xiang)需(mo)要(yu)的大佬可以玩玩~

### 1. 安装
```
npm install nativefier -g
```

### 2. 使用
```
 nativefier "https://www.zhihu.com/"
```
